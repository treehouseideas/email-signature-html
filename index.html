Prepare the HTML signature file:

  1. Click "Generate HTML" next to signature.
  2. Click "Show HTML".
  3. Your browser should open a new window with the HTML signature.
  4. Save this new window ('File' -> 'Save As') as a HTML file.

Instructions for Outlook:

Create a blank signature in Outlook:

  1. Go to "Tools" menu and select "Options" (if you don't have a "Tools" menu, go to "File" -> "Options)
  2. Go to "Mail Format" tab (alternatively, "Mail" option on the left)
  3. Click "Signatures" button
  4. Click "New" button
  5. Enter "netreca-signature"for the name of your signature
  6. Click "OK" to save the new signature
  7. Click "OK" again to close signature editing window
  8. Click "OK" again to close options window

Install our custom signature: Find Outlook signatures folder:

  1. Click Start
  2. In the "Run" input (it's normally the little input box at the bottom of start menu), enter "%APPDATA%" and hit "Enter" (alternatively, if you don't have that option in start menu you can start it with keyboard shortcut Windows+R)A folder should open. If it didn't, try doing the same and entering "%APPDATA%\" into the input box.
  3. In the folder that opened, if you have a "Microsoft" folder - open it. Otherwise, if you have "Roaming" folder - look in it, there should "Microsoft" folder in it.
  4. In "Microsoft" folder, open "Signatures" folder".
  5. Copy the saved HTML file into that folder (from your downloads folder, should be open since step 1 of this guide).

Set our custom signature as default:

  1. Go to "Tools" menu and select "Options" (if you don't have a "Tools" menu, go to "File" -> "Options)
  2. Go to "Mail Format" tab (alternatively, "Mail" option on the left)
  3. Click "Signatures" buttonYou should see your fancy new signature now.
  4. On the right side of this window, under "Choose default signature" select "NRG_signature" in both "new messages" and "replys/forwards" dropdowns.
  5. Click "OK" to save
  6. Click "OK" again to close options window

Test to make sure it works:

  1. Create a new email
  2. Verify your fancy new signature is already included
  3. Congrats! You should have the email signature show by default.

Instructions for Gmail:

Adding the signature to Gmail:

  1. Click "Select this signature" next to desired one
  2. Copy with Ctrl+C (or Cmd+C on Mac)
  3. In your gmail, click the rotary icon (right below profile picture)
  4. Click "settings" from drop-down menu
  5. Scroll down to "Signature" section
  6. Click in the text editor box for signature
  7. Paste with Ctrl+V (or Cmd+V on Mac), or right-click -> "Paste"
  8. Scroll to the bottom of the page and click "Save"
  9. Click "Compose" to verify your new signature works.
